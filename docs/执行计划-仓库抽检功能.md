## 技术约束

- 纯静态前端，零构建步骤
- 第三方库本地化在 `libs/`，不使用 CDN
- IndexedDB 单事务保证数据一致性
- 所有 UI 文案和注释为中文
- HTML 转义统一使用 `Utils.esc()` 防 XSS
- 摄像头需要 HTTPS 或 localhost 环境
- `sw.js` 的 `VERSION` 每次修改前端文件后必须递增

## 数据模型

### outbound-index.json
```json
{
  "version": "2026-04-14",
  "source": "SIT IT.xlsx",
  "count": 123,
  "codes": ["JY2502012"],
  "plainCodes": ["2502012"]
}
```

### inspections (IndexedDB)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | string | 主键 |
| rawCode | string | 扫码原始文本 |
| normalizedCode | string | 标准化后（JY2502012） |
| plainCode | string | 纯数字码（2502012） |
| result | string | `normal` / `warning` / `invalid` |
| message | string | 结果描述 |
| matchedSource | string | 命中的索引文件名 |
| scannedAt | string | ISO 时间 |
